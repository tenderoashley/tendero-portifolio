# 🌐 Personal Portfolio Website

## 📌 Project Overview

This project is a fully responsive personal portfolio website developed using HTML5 and CSS3. It showcases my skills, projects, and contact information in a clean and user-friendly interface.

The website follows modern web development practices including semantic HTML, mobile-first design, and responsive layouts.

---

## 🎯 Objectives

* To design and develop a professional personal portfolio
* To demonstrate understanding of HTML5 and CSS3
* To implement responsive web design
* To showcase personal projects and skills

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Visual Studio Code

---

## ✨ Features

* Fully responsive design (mobile, tablet, desktop)
* Navigation bar across all pages
* Home page with introduction
* About page with skills and background
* Projects page displaying work using CSS Grid
* Contact form with HTML5 validation
* Clean and structured layout

---

## 📱 Responsiveness

The website uses a mobile-first approach and adapts to different screen sizes using CSS media queries.

---

## 🧪 Testing & Debugging

* Tested using browser developer tools
* Debugged using Live Server in Visual Studio Code
* Validated HTML and CSS using W3C standards

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the folder in Visual Studio Code
3. Right-click on `index.html`
4. Select "Open with Live Server"
